CSS Variable

การสร้างตัวแปรใน css เพื่อช่วยอํานวยความสะดวกในการกําหนด style ให้ แต่ละ element โดยลดขั้นตอนการทํางานที่ซ˵าซ้อนใน css ให้โค้ดมีความเป็น ระเบียบและง่ายต่อการจัดการมากยิ่งขึ้น

การกําหนดตัวแปรสามารถกําหนดชื่อได้เอง (custom variable) โดยขึ้นต้น ด้วยเครื่องหมาย - ตามด้วยชื่อตัวแปร ซึ่งตัวแปรส่วนใหญ่ใน css จะนํามาเก็บค่าที่ ใช้เรียกทํางานซ˵าๆ เช่น สี เงา แอนิเมชั่น เป็นต้น
