การจัดตําแหน่งด้วย Position

position ใช้ในการจัดตําแหน่งของวัตถ

static - จัดวางแบบปกติเป็นค่า default
relative - ใช้ในการจัดวางและกําหนดตําแหน่งวัตถุโดย การนับจากจุดที่วัตถุนั้นๆอยู่
absolute - ใช้ในการจัดวางวัตถุ ให้ไปอยู่ตําแหน่งใดก็ได้แต่ต้องกําหนด ตําแหน่งจากขอบของ elememt ที่บรรจุ วัตถุนั้นๆอีกที (ถ้าไม่มีอะไรครอบก็ ถือว่า body เป็น elememt ที่ครอบวัตถุที่ระบุ absolute)
fixed - ใช้ในการจัดวางวัตถุให้อยู่ตําแหน่งเดิม
sticky - ให้วัถตุติดขอบเมื่อเลื่อนไปถึง
