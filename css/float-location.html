กําหนดตําแหน่งด้วย float

float กําหนดให้อิลิเมนต์สามารถลอยอยู่ด้านใดด้านหนึ่ง โดยค่าที่กําหนดได้ คือ

- left, right (ให้ลอยอยู่ทางด้านซ้ายและขวาตามลําดับ)
- inherit ลอยตาม parent element
- none ไม่ให้มีการลอย (default)

clear ไม่อนุญาตให้มีการลอยของอิลิเมนต์โดยค่าที่กําหนดได้คือ

- left, right (ไม่อนุญาตให้มีลอยทางด้านซ้ายและขวาตามลําดับ)
- both (ไม่อนุญาตให้มีการลอยทั้งสองด้าน)
